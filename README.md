# Degradation-Aware Image Restoration

A student project exploring different image restoration techniques — from classical filtering methods in Python to MATLAB-based dehazing and out-of-sample restoration (OSR).

---

## Overview

This project implements and compares multiple image restoration approaches on degraded images. It's organized into three modules:

- **Python pipeline** (`main.py` + `model.py`) — takes an input image and runs it through several processing techniques including Gaussian, median, and bilateral filtering, Sobel and Canny edge detection, Otsu thresholding, and morphological operations. All results are saved individually and as a combined comparison grid. We also did a quick check in Python to see if OSR could be implemented here, but it was mainly exploratory — the actual OSR implementation lives in MATLAB.
- **OSR module** (`OSR NEW/`) — a MATLAB implementation of Out-of-Sample Restoration using wave function-based reprogramming to recover image quality.
- **Dehazing module** (`Rehaze/`) — a MATLAB script that removes haze from a given input image and saves the restored output.

---

## Motivation / Problem Statement

Real-world images frequently get degraded — due to noise, atmospheric haze, blur, or poor lighting. Restoring such images matters a lot in fields like medical imaging, surveillance, and remote sensing.

We wanted to study and compare how different restoration methods perform on the same image, and also get hands-on with the OSR approach, which looks at restoration from a more generalized, research-oriented angle. This project helped us understand both the classical and modern sides of image restoration.

---

## Key Features

- Runs **multiple restoration and filtering techniques** in one go and saves all results for comparison
- **Morphological operations** (opening and closing) alongside standard filters
- **Edge detection** using both Sobel and Canny methods
- MATLAB-based **Out-of-Sample Restoration** (OSR) using wave function model reprogramming
- MATLAB-based **single image dehazing** with before/after outputs
- Clean folder structure — inputs and outputs are clearly separated

---

## Tech Stack

- **Python 3.x**
- **OpenCV** — image filtering, edge detection, thresholding, morphological ops
- **NumPy** — numerical operations on image arrays
- **Matplotlib** — generating and saving result comparison grids
- **MATLAB** — OSR restoration and dehazing scripts

## How to Run the Project

### Python Pipeline

**Step 1 — Clone the repo**

```bash
git clone https://github.com/Vidit-guptaa/degradation_aware_restoration.git
cd degradation_aware_restoration
```

**Step 2 — Install dependencies**

```bash
pip install -r requirements.txt
```

**Step 3 — Add your image**

Place your image inside the `input/` folder. You can replace `sample.jpg` or update the path in `main.py`.

**Step 4 — Run**

```bash
python main.py
```

Outputs will be saved automatically to the `output/` folder. Check `all_results.png` for a side-by-side comparison of every technique.

---

### MATLAB — OSR Restoration

1. Open MATLAB and go to the `OSR NEW/` folder
2. Open `OSR_ImageRestoration_final.m`
3. Make sure `OSR_INPUT.jpeg` is in the same folder
4. Run the script — output will be saved as `OSR_OUTPUT.jpeg`

---

### MATLAB — Dehazing

1. Open MATLAB and go to the `Rehaze/` folder
2. Open `dehaze_single_image_final.m`
3. Set the input image path in the script (defaults to `Hazed.jpeg`)
4. Run the script — dehazed result saves as `Output.jpeg`
